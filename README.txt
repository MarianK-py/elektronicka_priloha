	- pridavanie_udajov(citany_subor, zapisovaci_subor,a, b)
	- parametre funkcie:
		- citany_subor - retazec - názov súboru obsahujúceho pacientov určených na spracovanie (aj s príponou)
		- zapisovaci_subor - retazec - názov súboru do ktorého sa získané dáta ukladajú (aj s príponou)
		- a, b - čísla - interval hárkou ktoré majú byť spracovávané
